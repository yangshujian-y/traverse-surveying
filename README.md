# 归档图片批量复制与页码工具

这是一个 Python 桌面小工具，用来处理归档图片：

- 选择两个指定文件，先完整复制源目录到输出文件夹，再投放到输出成果中所有第 3 级且名称以 `0005`、`0007` 结尾的文件夹中。
- 递归扫描源目录下的户文件夹，并按每户分别成册编号；每户内按文件夹名称顺序和图片名称顺序从起始页码连续编到最后，默认竖版和横版都使用右上角。
- `0005`、`0007` 和 `0011` 文件夹支持正反面规则：正面页码右上角，反面页码左上角。
- 区分横版、竖版、正面、反面的页码位置，可在预览图上拖拽框选，后续统一按框选位置编页码。
- 编页前可按电子版左上角装订规则自动校正横竖方向，原始图片不被修改。
- 输出投放统计、编页统计、文件夹数量统计，方便校核。

## 使用流程

界面分为几个页面：顶部选择路径，中间用标签页切换“0005/0007 投放”“图片编页”“预览框选”“统计”，底部固定显示运行日志和进度。中间工作区和日志区之间可以拖动调整高度；设置项页面可滚动。

1. 选择“源文件夹”：通常是包含每一户子文件夹的总目录。
2. 在“编号”里填写新文件名后缀，默认 `-0002`。
3. 如需按截图目录投放，选择“0005文件”和“0007文件”，然后点击“执行 0005 / 0007 投放”。
4. 选择“输出文件夹”：程序会在这里生成 `输出文件夹/源文件夹名称` 成果目录，不直接改源文件。
5. 如需设置页码位置，选择一张预览图片，然后在“预览框选”页拖拽框选。预览页支持放大、缩小、适应窗口，并可显示页码实际效果；修改字体大小、颜色或示例页码后，预览会自动刷新。
6. 在“框选位置”里分别设置：
   - `portrait_front 竖版正面右上角`
   - `portrait_back 竖版反面左上角`
   - `landscape_front 横版正面右上角`
   - `landscape_back 横版反面左上角`
7. 点击“复制 + 编页 + 统计”，或分别执行投放、编页、统计。

编页规则：程序会把路径里第一个名称包含 `-` 的文件夹识别为“户文件夹”，例如 `0084-TDQQ0220520300-0003`。每个户文件夹单独从起始页码编号；户内先按子文件夹名称顺序，再按图片名称顺序连续编号到最后，不会在户内每个小文件夹重新从 `1` 开始。界面里的“备用成册级数”只在未识别到户文件夹时使用。普通文件夹内图片统一使用正面位置；只有图片直接所在的文件夹名称后缀为 `0005`、`0007` 或 `0011` 时才按正反面处理，祖先文件夹名称不会影响下级图片。左右位置按每个双面文件夹内部重新计算：该双面文件夹第 1、3、5 张使用正面位置，第 2、4、6 张使用反面位置，不会延续前一个单页文件夹的奇偶。字体大小统一使用“字体大小”输入框。

“并行线程数”控制复制和编页时的并发数量，默认 4。图片很多时可以适当调大，例如 6 或 8；电脑变卡时调小。

预览框选时建议只框住页码实际占用的小区域，避免页码压盖表格或正文。勾选“显示页码效果”后，预览图会直接显示示例页码，便于微调位置。

“编页前智能擦除四角旧页码”开启后，程序会在写入新页码前扫描图片四个角，自动擦除疑似旧页码的小型深色文字块，再写入新页码。

“按装订规则自动校正横竖方向”开启后，程序会在输出成果里自动旋转方向错误的图片：A3 按正文北侧短边装订，保持横向；A4 竖版按左边装订，保持竖向；A4 横版按正文北侧长边装订，保持横向；小于 A4 按正文左侧短边装订，保持竖向。A3/A4/小于 A4 会按同批图片像素面积相对判断，处理结果会写入 `编页统计.csv` 的 `page_layout` 和 `orientation_action` 字段。

0005/0007 投放规则：程序会先把整个源目录复制到输出文件夹内的同名成果目录，原文件保持不变；然后遍历所有第 3 级、目录名最后 4 位是 `0005` 或 `0007` 的文件夹。例如 `二级目录/三级户目录/0084-TDQQ0210120100-0001-0005` 会收到“0005文件”；第 2 级或第 4 级同后缀文件夹不会处理。

复制源目录到成果目录时，会同步文件夹内所有文件，包括图片。若成果目录已有同名文件但源文件更新过或大小不同，会自动覆盖更新，避免旧成果漏图或保留旧图。

源文件夹必须选择原始目录，不要选择名称带 `_输出` 的成果目录；输出文件夹也不要放在源文件夹内部。编页默认会递归扫描源文件夹下所有子文件夹，包括第三级文件夹下面的子文件夹；若仍未发现图片，程序会在日志中提示，编页不会继续处理。

投放进去的新文件会命名为 `目标文件夹名称-0002.扩展名`。例如复制到 `0084-TDQQ0210120100-0001-0005` 文件夹内时，新文件名是 `0084-TDQQ0210120100-0001-0005-0002.jpg`。

`0005-0007投放统计.csv` 会记录每个目标文件夹放入前文件数量、放入后文件数量、数量变化和复制状态。

执行过程中底部会固定显示实时日志。日志会显示开始、扫描数量、当前处理进度、统计文件生成位置，以及最终是否完成。
程序会在点击执行时先固定本次输入参数，后台处理期间不会再直接读取界面控件，避免 macOS 上长任务执行时界面线程异常退出。

## 输出文件

输出文件夹中会生成：

- `编页统计.csv`
- `文件夹数量统计.csv`
- `总处理统计.csv`

CSV 使用 `utf-8-sig` 编码，Excel 直接打开也能正确显示中文。
